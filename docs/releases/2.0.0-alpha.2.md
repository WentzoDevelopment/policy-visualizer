# Policy Visualizer Release Notes — 2.0.0-alpha.2

## Release

- Version: `2.0.0-alpha.2`
- Release date: `2026-02-24`
- Release type: `alpha`
- Commit/tag: _(to be tagged after commit)_

## Summary

Export Foundation milestone. Adds one-click PNG and PDF export of the React
Flow diagram exactly as it appears on screen. Export is entirely client-side
(no new backend route) using `html-to-image` and `jsPDF`. The full diagram is
always captured regardless of the user's current zoom or pan state.

## Highlights

- "Export PNG" and "Export PDF" buttons in the top-right panel of the diagram
- Capture always shows the full graph (fitView applied before capture, viewport restored after)
- UI chrome (controls, minimap, export panel) is excluded from the captured image
- Buttons disable during export to prevent double-clicks

## Scope for This Release

- Included:
  - EXP-210: Client-side PNG/PDF export via `html-to-image` + `jsPDF` in `FlowDiagram.tsx`
